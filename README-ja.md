# 🌈 Rainbow Tree

**Obsidian のファイルエクスプローラーを虹色に彩るプラグイン。**

フォルダやファイルにパレットの色を自動で割り当て、ネストされたアイテムには左端に祖先フォルダの色バーを積層表示します。フォルダの深さが一目で分かります。

![Rainbow Tree スクリーンショット](https://raw.githubusercontent.com/satoshikokubo/rainbow-tree/main/screenshot.png)

## 特徴

- **レインボー背景** — 7色パレット（赤→橙→黄→緑→青→藍→紫）で自動着色
- **階層バー** — 子アイテムの左端に、祖先フォルダの色が積み重なるバーを表示
- **ダーク／ライトテーマ対応** — テーマ切り替えで色が自動で変わる
- **2つの色モード** — *安定*（パスベース、常に同じ色）と *動的*（隣接する行の色重複を最小化）
- **パレット完全カスタマイズ** — 色の追加・削除・変更が可能（ダーク／ライト個別指定）
- **フォルダごとの色指定** — 右クリックで特定フォルダにカスタム色を割り当て
- **バースタイル調整** — 幅（2〜16px）、不透明度（0.1〜1.0）、ギャップ（0〜4px）
- **スムーズアニメーション** — フォルダ展開時のフェードイン
- **日本語・英語対応**

## インストール

### コミュニティプラグインから（推奨）

1. **設定 → コミュニティプラグイン → 閲覧** を開く
2. **"Rainbow Tree"** で検索
3. **インストール** → **有効化**

### 手動インストール

1. [Releases](https://github.com/satoshikokubo/rainbow-tree/releases) から最新版（`main.js`, `manifest.json`, `styles.css`）をダウンロード
2. Vault内に `<vault>/.obsidian/plugins/rainbow-tree/` フォルダを作成
3. 3ファイルをコピー
4. **設定 → コミュニティプラグイン** で有効化

## 設定

| 設定項目 | デフォルト | 説明 |
|---------|---------|------|
| Rainbow Treeを有効化 | ON | プラグイン全体の有効/無効 |
| 色割り当てモード | 安定 | *安定*: パスベースで常に同じ色。*動的*: 隣接行の色重複を最小化 |
| バー幅 | 16 px | カラーバーの幅 |
| バー不透明度 | 0.25 | 背景色とバーの不透明度 |
| バー間ギャップ | 3 px | バー間の隙間 |
| 展開アニメーション | ON | フォルダ展開時のフェードイン |
| 行間を詰める | OFF | 行間を最小化（実験的、一部テーマと競合する可能性あり） |

## 色モード

### 安定モード（デフォルト）

パスのハッシュ値で色を決定。スクロールやフォルダ開閉で色が変わらない安定した表示。

### 動的モード

表示順で色を割り当て、隣接行の色重複を積極的に回避。視認性を優先するユーザー向け。

## カスタマイズのヒント

- **バー幅を太く**（12〜16px）すると、カラフルで存在感のある見た目に
- **不透明度を下げる**（0.1〜0.15）と、控えめで上品な色合いに
- **バーギャップを0**にすると、隙間のないフラットな表示に
- **フォルダを右クリック**して色を指定すると、重要なプロジェクトフォルダを目立たせられる

## 互換性

- Obsidian **v1.5.0** 以上
- デスクトップ（Windows, macOS, Linux）・モバイル（iOS, Android）
- ダーク／ライトテーマ両対応
- カスタムCSSテーマとも併用可能
